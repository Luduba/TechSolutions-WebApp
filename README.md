# TechSolutions - IT Consulting Web Application

A fully responsive, multi-page web application for **TechSolutions**, an IT consulting company. 



| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Landing page with hero section and services overview |
| About | `about.html` | Company information, mission, values, and statistics |
| Services | `services.html` | Detailed list of IT services offered |
| Portfolio | `portfolio.html` | Project showcase with filter functionality |
| Blog | `blog.html` | Company news, insights, and updates |
| Team | `team.html` | Team member profiles and expertise |
| FAQ | `faq.html` | Frequently asked questions |
| Contact | `contact.html` | Contact form with validation |
